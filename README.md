# PmuMentor
A peer mentorship platform for PMU students designed to connect experienced students (mentors) with newer students (mentees) to support accademic success, personal development, and career readiness.

This repository contains the prototype implementation of the PmuMentor system developed as part of a Software Engineering course project using Agile (Scrum) methodology.

PmuMentor enables students to:
- Create and manage user accounts
- Login securely to the system
- Search for relevent mentorship resources and events
- Access guidance related to academics, careers, and personal growth

This prototype focuses on implementing core functionalities to demonstrate system feasiblity and design.

Features implemented in this prototype:
1 - User registraition:
  Allows students to create an account using email, password, and basic profile details.
  This feature ensures that only registered can access the platform.
2 - User login:
  Validates user credentials and allows authenticated users to access the system.
  Only users with valid login information can proceed to use platform features.
3 - Search service:
  Allows users to perform full-text searches to quickly find relevant information accross available resources and events within the system.
